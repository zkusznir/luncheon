== Luncheon

Luncheon is a simple rails application for managing meal orders. After cloning the application install all gems with
* <tt>bundle install</tt>
Migrate the database
* <tt>rake db:migrate</tt>

The application is intended to be set up locally only so far, in order to run the tests, please run the rspec command on chosen tests, e.g. all feature tests:
* <tt>rspec spec/features/</tt>

Ruby version: 2.2.2
Rails version: 4.2.5.1
